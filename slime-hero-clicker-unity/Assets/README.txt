# Unity Project Structure: Slime Hero Clicker

This folder contains the core assets for your Unity project.
- Assets/Scenes: Put your .unity files here.
- Assets/Sprites: All 2D textures and sprites.
- Assets/Scripts: C# logic (Already populated).
- Assets/Prefabs: Reusable game objects (e.g., Slime, FloatingText).
- Assets/Audio: Music and sound effects.
- Assets/Fonts: TextMeshPro fonts.
